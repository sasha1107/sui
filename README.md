# Welcome to SUI

React UI Library

## Installation

```sh
npm install suhyun-ui
```

## Usage

```jsx
import { Button, Header } from "suhyun-ui";

function App() {
  return (
    <div>
      <Header>
        <h1>Hello Suhyun UI</h1>
      </Header>
      <Button>Button</Button>
    </div>
  );
}
```
